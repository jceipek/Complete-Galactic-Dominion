init()

source code 

Game initialization function.

  1. Creates a Debugger (debugger) and EventTimer (eventTimer) and stores references to them in an instance of EventManager.
  2. Several listeners are started and registered with eventManager:
    • Instance of Universe (universe)
    • Instance of UserInterface (ui)
    • Instance of Window (gameWindow)
  3. We send the eventManager a message to start the game
    • This message is interpreted by the gameWindow
