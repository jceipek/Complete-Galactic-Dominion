Class Viewport

source code

Acts as a "window" into the world it contains. Includes deadzone dimensions so that it can be scrolled with the mouse.

#Attributes: # world = the world the viewport can see # scrollLoc = (x,y) #World coordinates of the corner of the viewport # loc = (x,y) #Corner of the viewport in screen coordinates # size = (width,height) #Dimensions in screen coordinates # mouse = the mouse controlling it # deadZoneRect
